# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1**.

Full text: https://www.contributor-covenant.org/version/2/1/code_of_conduct/

## Reporting

Report unacceptable behavior to: **d9ng@outlook.com**

Reports are reviewed confidentially. Response target: within 7 days.

## Scope

Applies to all project spaces — issues, pull requests, discussions, and any public channel where contributors represent the project.
